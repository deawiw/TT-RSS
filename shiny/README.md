# Shiny Dashboard для TT-RSS Analyzer

Интерактивный дашборд для визуализации нормализованных новостных данных.

## Функционал
- **Обзор данных**: KPI-карточки, графики по темам и источникам, таблица всех статей
- **Темы**: фильтрация и просмотр статей по категориям
- **Экспорт**: скачивание данных в CSV

## Требования
- R >= 4.0
- Пакеты: shiny, shinythemes, dplyr, ggplot2, plotly, DT
